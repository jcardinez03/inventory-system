# Store Management System

**Laravel | MySQL | Bootstrap | JavaScript**

A web-based system for managing inventory, sales, stock movements, audit logs, and basic business reports for small stores.

## Features

* User authentication
* Product management
* Category management
* Stock-in and stock-out tracking
* Stock movement history
* Low stock monitoring
* Dashboard summary
* Sales management *(planned)*
* Basic accounting reports *(planned)*
* Audit logs *(planned)*
* Deployed on Railway

## Tech Stack

* Laravel 11
* PHP 8.2
* MySQL
* Blade
* Bootstrap
* JavaScript
* Railway

## Purpose

This project was built as a portfolio project to practice Laravel, database relationships, CRUD operations, authentication, stock management, and business workflow logic.

The system started as an inventory management project and is planned to expand into a more complete store management system with sales, basic accounting, auditing, and reporting features.
