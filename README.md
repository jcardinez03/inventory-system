# Inventory Management System

A Laravel-based inventory system for managing products, categories, stock-in, stock-out, and stock movement history.

## Features

- User authentication
- Product management
- Category management
- Stock-in and stock-out tracking
- Stock log history
- Low stock monitoring
- Dashboard summary
- Deployed on Railway

## Tech Stack

- Laravel 11
- PHP 8.2
- MySQL
- Blade
- Bootstrap
- JavaScript
- Railway

## Purpose

This project was built as a portfolio project to practice Laravel, database relationships, CRUD operations, authentication, and inventory logic.
